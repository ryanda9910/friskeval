# Changelog

All notable changes to friskeval are documented here.

## [0.1.0] — 2026-07-09

First release.

### Added
- The skill (`skill/SKILL.md`) — Builds a TF-IDF model of every skill description; flags near-duplicate pairs, security skills that claim another's domain, and trigger prompts that route to the wrong owner..
- One-line installers (`install.sh`, `install.ps1`); multi-harness.
- Docs (`docs/`), real runs (`CASES.md`), i18n (EN / ID / zh-CN), key-free demo.
- MIT license, CONTRIBUTING, commitlint config + CI + opt-in hook.
